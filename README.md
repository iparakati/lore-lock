# Lore Lock - Prison Break (Inform 7)

This project has been converted from a Python-based engine to an Inform 7 interactive fiction game.

## Overview

"Prison Break" is a short text adventure where you must escape a damp stone cell.

## How to Play

1.  Download the `prison_break.ni` file.
2.  Open the file in the [Inform 7 IDE](http://inform7.com/).
3.  Click "Go" or "Release" to play the game.

## Testing

The source file includes a test command. inside the game type:

    test me

This will run a sequence of commands to verify the game can be completed.
